# 🛣️ 道路裂缝检测系统

基于 U-Net 深度学习模型的道路裂缝自动检测系统

## 📋 项目简介

这是一个使用深度学习技术自动检测道路裂缝的项目。通过训练 U-Net 神经网络模型，系统可以自动识别道路图片中的裂缝位置，并生成可视化结果。

### 什么是道路裂缝检测？

道路裂缝检测是指通过计算机视觉技术，自动识别道路表面的裂缝。传统方法需要人工巡检，费时费力。本项目使用深度学习自动完成这一任务。

### 什么是 U-Net？

U-Net 是一种专门用于图像分割的深度学习网络结构。它像一个"U"字形，可以精确地识别图片中每个像素点的类别（背景或裂缝）。

## ✨ 主要功能

- ✅ **自动检测**：上传道路图片，自动识别裂缝位置
- ✅ **可视化结果**：生成裂缝标注图和叠加可视化
- ✅ **裂缝量化**：计算裂缝占比和严重程度
- ✅ **Web界面**：提供友好的图形界面，无需编程知识
- ✅ **跨平台**：支持 Linux 和 Windows 系统

## 🚀 快速开始

### 前置要求

- Python 3.7 或更高版本
- （可选）NVIDIA GPU + CUDA（用于加速训练）

### 步骤 1: 克隆项目

```bash
git clone https://github.com/Makeprograss/U-net_to_check_road_crake.git
cd U-net_to_check_road_crake
```

### 步骤 2: 安装依赖

```bash
pip install -r requirements.txt
```

### 步骤 3: 准备数据

将你的道路图片和标注放入 `data/` 文件夹。详细说明请查看 [数据准备指南](./docs/01_数据准备指南.md)

### 步骤 4: 运行应用

**使用预训练模型（推荐初学者）：**

```bash
# Linux / Mac
bash run_linux.sh

# Windows
run_windows.bat
```

**从头训练模型：**

```bash
python model_train.py
```

详细说明请查看 [模型训练指南](./docs/02_模型训练指南.md)

## 📊 技术指标

- **模型**: U-Net
- **输入尺寸**: 224 × 224
- **输出**: 二分类分割（背景/裂缝）
- **评估指标**: IoU (Intersection over Union)
- **训练数据集**: 自定义道路裂缝数据集

## 🔧 系统要求

### 最低配置

- CPU: 双核处理器
- 内存: 4GB RAM

## ❓ 常见问题

### Q: 我没有 GPU，能训练模型吗？
A: 可以，但训练速度会较慢。推荐使用预训练模型或使用 Google Colab 等免费 GPU 资源。

### Q: 如何获取训练数据？
A: 可以使用公开数据集（如 CrackForest, DeepCrack）或自己拍摄道路图片并标注。

### Q: 检测精度不高怎么办？
A: 可以尝试：
1. 增加训练数据
2. 调整训练参数
3. 使用数据增强
4. 训练更多轮次

详细解决方案请查看 [模型训练指南](./docs/02_模型训练指南.md)
